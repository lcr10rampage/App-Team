# Engineering Memory — Team Index

> **Seeded, authoritative source:** `app-build-set/EngineeringMemory.md`. That file (built from real projects) is the primary engineering knowledge base — proven patterns, lessons, common mistakes, design principles, architecture decisions. **Do not duplicate it here.**
>
> This file is the team-system-level extension: it holds durable, cross-project engineering knowledge produced *by the team process* (retrospectives), and points to the foundation for everything already captured.

## How this file is maintained
- Only the **Manager** approves entries (see `app-build-set/engineering-knowledge-prompt.md` for the capture standard and the ★ confidence scale).
- Entries come from the retrospective (`templates/retrospective-template.md`) after RELEASE_APPROVED.
- Keep it curated, not an activity log. Merge/duplicate-check periodically.
